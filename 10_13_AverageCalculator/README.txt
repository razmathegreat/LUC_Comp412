The Average calculator is a program that takes the values from  IntegerInput.txt and outputs the average of these values in the command line. 
You can edit this file to change the integers to your desire. Inputting non integer values will cause this code to not run.

*****TO RUN*****
1. Download Repo
2. Open a command line and set the directory to the src folder of the repository.
3. Type  java -jar  AverageCalculator.jar
